# lifeos

Planner App

A lightweight planner app built with plain HTML, CSS, and JavaScript. It lets you:

- add tasks with a due date, time, priority, and notes
- mark tasks as complete
- delete tasks you no longer need
- keep plans saved in the browser with local storage

## Running locally

Open `/home/runner/work/lifeos/lifeos/index.html` directly in a browser, or serve the repository with a simple static file server.
